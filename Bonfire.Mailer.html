        View Source Bonfire.Mailer (Bonfire v0.9.10-classic-beta.136)

        An library for Bonfire that contains:

        • Sending email using Bamboo, which supports SMTP and a multitude of email delivery providers (some of which require an extra dependency)
        • Checking hypothetical email or domain validity

        send(email, to, mode \\ :async, opts \\ [])

        View Source

        Sends an email with specified mode (immediate or async).

        Parameters

        • email: The email content to be sent.
        • to: The recipient's email address.
        • mode: The sending mode (:now or :async).
        • opts: Optional parameters for sending the email.

        Examples 

        iex> Bonfire.Mailer.send("Hello", "recipient@example.com", :now)
{:ok, %Bamboo.Email{}}
